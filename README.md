# **HDD Simulator v5.0**

Web-based Horizontal Directional Drilling simulator by Julio Sarracent.

### **Features**

* 20 precision levels.  
* Real-time telemetry & Tool Face.  
* Global leaderboard (Firebase).  
* Safety: 811 & Potholing.

### **Tech**

HTML5, Tailwind, JS Canvas.

Conceptual training tool with industrial UI.
